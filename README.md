# Sales & Performance Analysis
**Tools:** SQL · Power BI | **Author:** Mohammad Danish

## Overview
Interactive B2B sales dashboard tracking ₹4M revenue, 55.69% profit 
margin across 4 regions and 6 sales reps.

## Key Insights
- North region leads revenue at ₹1.5M
- Software category dominates at 33.61% revenue share
- Ayesha Khan is top sales rep
- Average profit margin: 55.69%

## Files
- `01_schema_and_data.sql` — 6-table schema + 45 orders seed data
- `02_analysis_queries.sql` — 10 analysis queries (revenue, profit, QoQ growth)
- `Sales_Performance_Data.xlsx` — Dataset connected to Power BI
